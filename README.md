# ✊✌️🖐️ 猜拳遊戲 (Rock Paper Scissors)

A sleek, cyberpunk-styled Rock Paper Scissors game built with vanilla HTML/CSS/JavaScript. No dependencies, no build step — just open and play.

## 🎮 Live Demo

👉 [Play on Vercel](https://janken-game.vercel.app)

## ✨ Features

- 🎨 Cyberpunk dark UI with glow effects
- 📊 Real-time scoreboard (Win / Draw / Lose)
- 🤚 Animated hand gestures with shake effect
- 📱 Fully responsive (mobile-friendly)
- ⚡ Zero dependencies — pure HTML/CSS/JS

## 🚀 Quick Start

Just open `index.html` in any browser — no setup required.

```bash
# Or serve locally
npx serve .
```

## 🕹️ How to Play

1. Click **石頭 (Rock)**, **剪刀 (Scissors)**, or **布 (Paper)**
2. The computer randomly picks its move
3. Score updates automatically
4. Click **重置分數** to reset the scoreboard

## 📋 Rules

| You | Computer | Result |
|-----|----------|--------|
| ✊ Rock | ✌️ Scissors | 🎉 You Win |
| ✌️ Scissors | 🖐️ Paper | 🎉 You Win |
| 🖐️ Paper | ✊ Rock | 🎉 You Win |
| Same | Same | 🤝 Draw |
